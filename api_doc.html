1. POST /signup?login=kek&password=kek&name=kek&surname=kek&patronymic=kek - регистрация пользователя
    Возвращаемые значения:
    • status 409 'Not signed up. Login already exists.'
    • status 201 'Signed up.'
    • status 500 'Not signed up. Internal error.'
  2. GET /signin?login=kek&password=kek - вход пользователя
    Возвращаемые значения:
    • status 202 'Signed in.'
    • status 401 'Not signed in. Invalid password.'
    • status 401 'Not signed in. No such user.'
  3. GET /user_groups?login=myLogin - группы пользователя
    Возвращаемые значения:
    • status 200 json: {"amount":2,"groups":[{"id":"1","name":"keek"},{"id":"2","name":"test"}]}
    • status 400 'No such user.'
  4. GET /find_users?key=m - поиск пользователей по префиксу без учета регистра
    Возвращаемые значения:
    • status 200 json: {"amount":2,"users":["myLogin","mark_zuckerberg"]}
  5. GET /users_in_group?group_id=1 - пользователи в группе
    Возвращаемые значения:
    • status 200 json: {"amount":2,"users":["admin","kek"]}
  6. GET /find_group?group_id=1 - имя группы по id
    Возвращаемые значения:
    • status 200 group_name
    • status 400 'No such group.'
  7. POST /new_group?group_name=testGroup&group_admin=kek - создание новой группы. group_admin - создатель
    Возвращаемые значения:
    • status 200 group_id
    • status 500 'Group was not created.'
  8. POST /add_users_to_group?members[]=kek&members[]=admin&group_id=1 - добавление списка пользователей вгруппу
    Возвращаемые значения:
    • status 200 'Users were added.'
    • status 500 'Users were not added.'
  9. POST /new_fav?to=userTo&from=userFrom - новая заявка на добавление в избранное.
    Возвращаемые значения:
    • status 405 'Already waiting for confirmation.'
    • status 200 'Fav was added.'
    • status 500 'Fav was not added.'
  10. GET /unconfirmed_favs?login=kek - Список неподтвержденных входящих заявок пользователя
    Возвращаемые значения:
    • status 200 JSON: {"amount":2,"users":["admin","kek"]}
  11. POST /confirm_fav?to=userTo&from=userFrom - подтверждение заявки на добавление в избранные
    Возвращаемые значения:
    • status 405 'No such request.'
    • status 200 'Fav confirmed.'
    • status 500 'Fav was not confirmed.'
  12. GET /favs?login=user - список подтвержденных избранных
    Возвращаемые значения:
    • status 200 JSON: {"amount":2,"favs":["admin","kek"]}
